# 🎥 Real-Time Video Streaming Scheduler (RTVSS)

A Python-based simulation of a real-time video streaming system using OS scheduling algorithms.

## 🚀 Features

* Real-time simulation of multiple video streams
* Scheduling Algorithms:

  * Earliest Deadline First (EDF)
  * Round Robin
  * Priority Scheduling
* Adaptive Bitrate (ABR) system
* Live performance metrics:

  * CPU usage
  * Latency
  * Bandwidth
  * Frame drops
* Gantt chart visualization

## ⚙️ Tech Stack

* Python
* Tkinter (GUI)
* Matplotlib (Visualization)
* Multithreading

## 📊 Concepts Covered

* OS Scheduling Algorithms
* Real-Time Systems
* Bandwidth Allocation
* Adaptive Bitrate Streaming
* Queue & Thread Management

## ▶️ How to Run

```bash
pip install -r requirements.txt
python rtvss_app.py
```

* Simulate network congestion
